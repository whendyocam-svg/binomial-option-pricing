# binomial-option-pricing
# 📈 Binomial Option Pricing Model

Este proyecto implementa el modelo binomial de Cox-Ross-Rubinstein para la valuación de opciones financieras.

## Características

* Opciones europeas y americanas
* Call y Put
* Comparación con modelo Black-Scholes
* Implementación eficiente (sin matrices grandes)

##  Tecnologías

* Python
* NumPy
* SciPy
* Matplotlib

## Ejemplo

Precio Binomial vs Black-Scholes:

```
Precio Binomial: 10.45
Precio Black-Scholes: 10.43
```

## Cómo usar

```bash
pip install -r requirements.txt
python main.py
```
